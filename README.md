# 🎮 Games Collection - JavaScript Games

Una colección de juegos clásicos implementados con HTML5, CSS3 y JavaScript vanilla.

## 📁 Estructura del Proyecto

```
Games-js/
├── snake/              # 🐍 Snake Game - 2 Players
│   ├── index.html      # Juego principal
│   ├── style.css       # Estilos modernos
│   ├── script.js       # Lógica del juego
│   └── README.md       # Documentación específica
├── .gitignore          # Archivos ignorados
└── README.md           # Este archivo
```

## 🎯 Juegos Disponibles

### 🐍 Snake - 2 Players
- **Ubicación**: `snake/`
- **Jugadores**: 2 simultáneos
- **Controles**: Flechas (Jugador 1) / WASD (Jugador 2)
- **Características**: Tres frutas, diseño moderno, responsive

## 🚀 Cómo Jugar

1. Navega a la carpeta del juego deseado
2. Abre `index.html` en tu navegador web
3. ¡Disfruta!

## 🛠️ Tecnologías

- **HTML5**: Estructura de los juegos
- **CSS3**: Estilos modernos y animaciones
- **JavaScript**: Lógica de juegos y renderizado

## 🤝 Contribuir

¡Las contribuciones son bienvenidas! Si quieres agregar un nuevo juego:

1. Crea una nueva carpeta con el nombre del juego
2. Implementa tu juego usando HTML/CSS/JS vanilla
3. Agrega un README.md específico en la carpeta del juego
4. Actualiza este README principal con la información del nuevo juego
