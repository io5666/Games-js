# 🐍 Snake Game - 2 Players

Un juego clásico de Snake implementado con HTML5, CSS3 y JavaScript vanilla, diseñado para dos jugadores simultáneos.

## 🎮 Características

- **Dos jugadores simultáneos**: Jugador 1 usa las flechas del teclado, Jugador 2 usa WASD
- **Tres frutas en el mapa**: Mayor desafío y oportunidades de puntuación
- **Controles intuitivos**: Las serpientes solo se mueven cuando presionas una tecla
- **Diseño moderno**: Interfaz visual atractiva con efectos de vidrio y animaciones
- **Responsive**: Se adapta a diferentes tamaños de pantalla

## 🎯 Cómo Jugar

1. **Inicia el juego** presionando el botón "Iniciar Juego"
2. **Jugador 1** (verde): Usa las flechas del teclado (↑↓←→)
3. **Jugador 2** (rojo): Usa las teclas WASD
4. **Objetivo**: Come las frutas amarillas para crecer y aumentar tu puntuación
5. **¡Cuidado!**: Evita chocar contigo mismo o con el otro jugador

## 🛠️ Tecnologías Utilizadas

- **HTML5**: Estructura del juego
- **CSS3**: Estilos modernos con gradientes, animaciones y efectos visuales
- **JavaScript**: Lógica del juego, controles y renderizado en canvas

## 🚀 Cómo Ejecutar

1. Abre `index.html` en tu navegador web
2. ¡Disfruta jugando!

## 🎨 Características Visuales

- **Fuente retro**: "Press Start 2P" para un look clásico de videojuegos
- **Efectos de vidrio**: Contenedor principal con backdrop-filter
